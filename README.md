# Code-OHMBASE 🚀

Selamat datang di repository resmi **Code-OHMBASE**. Repository ini berisi kumpulan kode sumber (source code) dari berbagai proyek teknologi, IoT, dan embedded systems yang dikembangkan oleh **Ohmbase**.

Semua kode yang ada di sini bersifat terbuka untuk dipelajari, diunduh, dan digunakan kembali untuk kebutuhan proyek Anda.

---

## 📁 Struktur Folder
Setiap folder dalam repository ini merepresentasikan satu proyek spesifik. Saat ini tersedia:
*   **read-ektp**: Kode untuk sistem pembacaan data e-KTP menggunakan sensor/modul terkait.
*   *(Folder lainnya akan segera ditambahkan sesuai perkembangan project)*

---

## 🛠️ Cara Menggunakan
Anda dapat menggunakan kode ini dengan dua cara:

## Download Langsung (ZIP)
1. Klik tombol hijau bertuliskan **Code** di bagian kanan atas halaman ini.
2. Pilih **Download ZIP**.
3. Ekstrak file di komputer Anda dan buka menggunakan IDE pilihan Anda (Arduino IDE, VS Code, dll).
